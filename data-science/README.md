# Data Science Study

Data Science notebooks covering statistics, machine
learning, and model deployment. Built on a foundation
of years of real-world DS work in airline revenue
management analytics.

**Domain strengths:**
Airline pricing · Demand forecasting · Biochemistry
· Health and life sciences data

---

## Stack being built

| Layer | Tools |
|-------|-------|
| Data manipulation | Pandas · NumPy |
| Statistics | scipy · statsmodels |
| ML | scikit-learn · XGBoost · LightGBM |
| Deep Learning | PyTorch · TensorFlow |
| NLP + LLM | HuggingFace · LangChain |
| Deployment | FastAPI · MLflow · Docker |
| Visualization | Matplotlib · Seaborn · Streamlit |
