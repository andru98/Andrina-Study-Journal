## How notebooks are built

Every notebook is recoded from memory after watching
the lecture — no copy paste. If it can't be retyped
from memory, the concept isn't owned yet.

Each notebook includes:
- Core concept explained in 2 sentences
- Working code recoded from memory
- Biochemistry or airline domain analogy
- Real-world DE use case connection
